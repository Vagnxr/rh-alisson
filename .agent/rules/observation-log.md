# Registro de Observacoes (Plataforma Financeira)

## Como usar
- Adicionar entradas novas no topo.
- Se a observacao virar regra permanente, mover para `docs/` ou `.cursor/rules`.

## Observacoes
- [Zustand] Nao desestruturar stores dentro de loops/callbacks.
- [TanStack Table] Configurar colunas via metadados.
- [Forms] Usar React Hook Form + Zod para validacao.
- [Mocks] Dados mockados devem representar contratos esperados da API.
- [Multi-loja] Sempre considerar filtro por loja e totais consolidados.
