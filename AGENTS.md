## Sobre o Projeto
Este e o frontend de uma **Plataforma Financeira & Operacional**, desenvolvido para substituir uma planilha financeira existente, mantendo suas regras, fluxo e logica, porem evoluindo para um sistema profissional, escalavel e seguro.

## Stack
- React + Vite + TypeScript
- Tailwind CSS + shadcn/ui
- Zustand (estado global)
- TanStack Table (tabelas dinamicas)
- React Hook Form + Zod (formularios e validacao)
