# Start Here

## Objetivo
Permitir que desenvolvedores consigam:
- Rodar o projeto localmente
- Entender as regras de negocio e os contratos
- Evoluir features sem introduzir divida tecnica

## Principios de Desenvolvimento
- Evitar over-engineering: priorizar entregas incrementais.
- Documentar decisoes (por que) antes de expandir escopo.
- Fazer features que fecham loop de valor: entrada -> processamento -> visualizacao.

## O que e a Plataforma Financeira
- Sistema para substituir planilha financeira existente
- Interface orientada a tabelas (semelhante ao Excel)
- Multi-tenant (isolamento por empresa)
- Multi-loja (consolidacao e visao individual)
- Frontend mockado inicialmente (validacao de UX)
